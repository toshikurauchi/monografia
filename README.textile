h2. Como compilar

No Windows execute no diretorio /monografia:

@compile.bat

Esse script executa os seguintes comandos (e necessario executar os 4 comandos por causa do bibtex):

@latex monografia@

@bibtex monografia@

@latex monografia@

@latex monografia@

Caso deseje converter para pdf:

@dvipdfm monografia@

Para a proposta.tex o processo e analogo
