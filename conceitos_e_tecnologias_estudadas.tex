\subsubsection{Classes}
Classes são definidas pelas palavras reservadas \textit{class} ou \textit{struct}, sendo que a diferença entre elas é o fato de atributos de uma \textit{class} serem privados por padrão e os de uma \textit{struct}, públicos. É possível adicionar modificadores de acesso através das palavras \textit{public}, \textit{private} e \textit{protected} que definem acesso livre em todos os escopos, acesso restrito à classe e acesso restrito à classe e suas subclasses respectivamente.

\subsubsection{Namespaces}
Para evitar colisão de identificadores como nomes de classes, variáveis e funções é possível utilizar um \textit{namespace}:

%Code
namespace nome {
// Entidades
    int n;
}
%/Code

Onde \textit{nome} é qualquer identificador válido e \textit{Entidades} é o conjunto de classes, variáveis e funções que ficarão protegidas dentro de um \textit{namespace}.

Para referenciar identificadores de um \textit{namespace} é utilizada a seguinte sintaxe:

%Code
<nome do namespace>::<identificador>
%/Code

Para acessar a variável \textit{n} do exemplo utiliza-se o seguinte código:

%Code
nome::n
%/Code
